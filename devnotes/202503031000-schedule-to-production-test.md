## テストアップ時の完了目標（3/11 時点）

### 必須

- [ ] TOPページが本番サーバーで表示できる
- [ ] ヘッダー・ヒーロー・フッターが仕様どおり動作する（TOP/下層・スクロール・スマホドロワー）
- [ ] Phase C 〜 J の全セクションが一通り実装済み（お知らせ〜お問い合わせ、見た目・レイアウト）
- [ ] レスポンシブ（PC・タブレット・スマホ）で崩れない
- [ ] 本番サーバーで SSL・ドメイン設定が可能な状態

### あればよい（後続でも可）

- [ ] お問い合わせフォームの送信処理（プラグイン導入等）
- [ ] 各セクションのデータを WordPress 管理画面から編集可能に
- [ ] ヒーロー複数スライド・カルーセル
- [ ] アクセシビリティ・パフォーマンスの最終調整
